# 이경근 / Gyeong Geun Lee

원자력 구조재료의 열화 및 신뢰성 평가와 관련된 데이터 수집, 처리, 모델링, 분석을 수행합니다.

## 소개

원자력 구조재료와 구조 건전성 분야의 도메인 지식을 바탕으로, 공학 데이터 분석과 반복 가능한 기술 문서화를 수행합니다. 주요 대상 재료는 압력용기용 저합금강, 내부구조물용 스테인리스강과 니켈 합금, 지르코늄 합금 압력관 등을 포함합니다.

## 관심 분야

- 원자력 구조재료의 열화 및 신뢰성 평가
- 원자로 압력용기, 내부구조물, 압력관 관련 데이터 분석
- R/Python 기반 데이터 분석
- 공학 데이터베이스와 기술 문서 관리

## 주요 작업

- 원자로 압력용기 및 내부구조물 관련 데이터 분석
- 재료 데이터베이스 및 정보 포털 개발
